Посилання на відео із покроковими діями:
https://youtu.be/FF6viTwK6H8
1. Відкрити Chrome DevTools
2. Зайти у панель Elements
3. Знайти інспектором кнопку Submit и вирівняти слово Submit по центру
4. Увімкнути режим мобільного девайсу (Device Mode)
5. Виставити Dimensions: Responsive 
6. Обрати Responsive - Mobile
7. Продемонструвати роботу дизайну Responsive
8. Виставити Dimensions: Adaptive (select device)
9. Створити свій девайс
10. Виставити Throttling low-end mobile
11. Відновити сторінку
12. Перевести в ландшафтний режим
13. Вимкнути Device Mode
14. Відкрити панель Console
15. Знайти 2 повідомлення Error в логах
16. Зберегти ці 2 повідомлення лога як файл на комп'ютері
17. Знайти 1 повідомлення verbose
18. Відкрити вкладку Sources
20. Відкрити файл Index
21. Відкрити вкладку Network
22. В url надіслати https://lnkd.in/dSJ_R7Yn
23. Знайти лог request/responce  
24. Відкрити вкладку Headers
25. Переглянути General, Response Headers, Request Headers
26. Знайти і скопіювати в новий текстовий файл header - User-Agent
27. Відкрити вкладку Preview
28. Раозгорнути всі рівні вкладення JSON
29. Відкрити вкладку Application.
30. Розгорнути все Storage в лівому стовпчику
31. Зайти в Cache Storage та Application Cache
32. Там всюди буде порожньо
33. Відкрити будь-який сайт, який хочете. (Увести url сайту в адресний рядок і натиснути Enter)
34. Відкрити вкладку LightHouse
35. Натиснути Generate report
36. Встановити в браузер безкоштовний AdBlocker
37. Встановити в браузер Json Viewer
38. Зайти на сайт https://ksendzov.com (https://ksendzov.com/)
39. Зробити і зберегти скріншот видимої частини сторінки сайту
40. Зробити і зберегти скріншот всього сайту
41. Змінити локацію GPS. Продемонструвати на сайті Гугл карт
42. Змінити локацію GPS другий раз. Продемонструвати на сайті Гугл карт 
43. Знайти інтерфейс для зміни User-Agent
44. Підставити інший User-Agent
45. Надіслати запит на https://lnkd.in/dSJ_R7Yn
46. Перевірити чи змінився ваш нативний User-Agent на новий
47. Створити свій юзер агент
48. Повторити пункти 45, 46
49. Знайти інтерфейс для зміни/додавання параметрів Throttling
50. Створити свої параметри поганого з'єднання і перевірити, як буде завантажуватися сторінка
51. Знайти інтерфейс для зміни/додавання параметрів Locations
52. Перевірити на Google карті як застосовуються зміни локації.
